== Description ==

A wordpress plugin which implements [Fancybox](http://fancybox.net/).

Fancybox does the same as lightbox, but in a much nicer way. Fancybox Plus plays videos from 
YouTube, blip.tv and Vimeo.

This plugin automatically enhance image links to use Fancybox. It has the same functionality as the wordpress plugin [Lightview Plus](http://wordpress.org/extend/plugins/lightview-plus/)

This plugin uses jQuery as Javascript Framework.
