== Description ==

Simple Auto Featured Image is a plugin that automatically generates the featured image for a post, page or custom post type if it does not have one.

The plugin automatically declares the featured image support with "add_theme_support" and lets you choose which custom post type can be managed by the plugin.

Translation: English, French 

== Installation ==

1. Upload the complete `simple-auto-featured-image` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Simple Auto Featured Image' under the 'Settings' tab and configure the plugin
