bitly - Do more with your links

== Description ==

*Bit.ly Service* replaces the default WordPress shortlinks with Bitly easily.

Adds links to the Global Stats/User's Stats page and allows you to share the link on Twitter.

= QR Code =

You need to give your `bitly-service/library/cache/` folder `755` permissions. If `755` doesn't work, try `777`.

`[qrcode url="http://bitly.com/WpbIlR" size="390"]`

= Bitly =

* [Create a free account](https://bitly.com/a/sign_up)
* [Your bitly API Key](https://bitly.com/a/your_api_key)

== Installation ==

1. Unzip the `bitly-service.zip` folder.
1. Upload the `bitly-service` folder to your `/wp-content/plugins` directory.
1. In your WordPress dashboard, head over to the Plugins section.
1. Activate Bit.ly Service.
